Bootstrap Landing Page

Saya pernah belajar dan membuat landing page dengan menggunakan framework Bootstrap. Landing page tersebut adalah untuk perusahaan bernama METATEL (Media Pratama Telekomunikasi), yang berfokus pada penyediaan layanan internet yang andal dan canggih. Dalam proses pembuatan, saya menggunakan berbagai komponen dan fitur dari Bootstrap untuk memastikan halaman memiliki tampilan yang modern, responsif, dan user-friendly.

PHP Sederhana

Setelah mempelajari pembuatan landing page dengan menggunakan Bootstrap, saya juga belajar mengenai PHP dasar dengan membuat formulir pemesanan tiket penerbangan.

OOP Sederhana

Setelah mempelajari PHP dasar, saya juga belajar tentang Object-Oriented Programming (OOP) dengan menggunakan PHP untuk membuat aplikasi yang lebih terstruktur dan modular. Berikut adalah deskripsi singkat dari proyek sederhana yang saya buat untuk mempelajari konsep OOP:

Sistem Pengelolaan Basis Data Mahasiswa

Proyek ini melibatkan pembuatan aplikasi berbasis web untuk mengelola data mahasiswa menggunakan operasi CRUD (Create, Read, Update, Delete). Tujuan utama dari sistem ini adalah menyediakan antarmuka yang mudah digunakan untuk pengelolaan data mahasiswa di dalam basis data universitas secara efisien.

KitaSehat: Wujudkan Indonesia Sehat

Proyek ini merupakan tugas akhir semester (UAS) yang berfokus pada pembuatan situs web dengan tema kesehatan. Tujuan dari proyek ini adalah untuk menciptakan platform yang dapat mengedukasi masyarakat mengenai kesehatan, memberikan akses ke layanan medis berkualitas, serta mendorong pola hidup aktif dan sehat.
